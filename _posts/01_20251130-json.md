title: Json basic info
---

1.可以把{}和[]分别当成结构体和数组，{}是object,[]是array  
2.一个json文件比如只有一个value，文件的最外层有两种情况，一个是{}，一个是[]，  
用对象作为根：  
'''
{
  "code": 200,
  "message": "ok"
}
'''
用数组作为根：  
'''
[
  {"id": 1}
  {"id": 2}
]
'''
以上两种都是合法的  
3.Json的根不能并列多个{}，例如在整个json文件中只有如下定义，是非法的，出现了两个根对象  
'''
{ "a": 1 }
{ "b": 2 }
'''
